# Assets Folder

This folder contains your personal files for the autofill extension.

## Required Files

1. **photo.png** - Your profile photo (PNG format)
2. **CV.pdf** - Your CV/Resume (PDF format)

## Setup Instructions

1. Add your photo as `photo.png`
2. Add your CV as `CV.pdf`
3. Run `node ../scripts/build-files.js` to generate `fileData.js`

## Privacy Note

⚠️ **These files are excluded from git** via `.gitignore` to protect your privacy.
Do not remove them from `.gitignore` if you plan to share this repository.
